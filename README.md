# Acme Relay

A compact TypeScript service used to exercise CodeRabbit Sales Replay. The repo intentionally contains a set of realistic product changes that were merged over time. Some issues are fixed by later commits on main and some remain open, so replay runs can demonstrate addressed-in-future behavior.

## Domains

- Billing and checkout
- Webhook ingestion
- Organization permissions
- Audit and exports
- Repository synchronization
- Account invitations and timezone formatting
